# 通知音效目录

此目录用于存放自定义通知音效文件。

## 使用说明

1. 将您的通知音效文件命名为 `notification.mp3` 并放置在此目录中
2. 支持的格式：MP3、WAV、OGG
3. 建议音效时长：1-2 秒
4. 建议文件大小：< 100KB

## 默认行为

如果未提供音效文件，系统将使用 Web Audio API 生成简单的通知提示音。

## 推荐音效来源

- [Freesound](https://freesound.org/) - 免费音效库
- [Mixkit](https://mixkit.co/free-sound-effects/) - 免费音效
- [Notification Sounds](https://notificationsounds.com/) - 通知音效专门站

请确保使用的音效符合相关版权许可要求。
